Step 1: Intake Request

Capture the request from client system. Verify mandatory details are present.

Audit Check: Ensure request ID is logged in Octopus.

Step 2: Validation

Validate member ID, date of service, and provider details.

Audit Check: Log validation outcome in Notes.

Step 3: Decision Flow

If data complete → proceed to review. If not → return request to sender.
